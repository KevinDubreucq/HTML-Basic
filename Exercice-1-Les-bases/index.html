Les balises

Les balises paires

Les balises paires reprennent la grande majorité des balises en HTML, elles fonctionnent par deux, une qui ouvre et une qui ferme. Voici quelques exemples:

  • la balise <h1></h1> qui permet de mettre en forme les titres,
  • la balise <p></p> qui permet de créer un paragraphe,
  • la balise <head></head> qui définit la tête d'un document,
  • la balise <title></title> qui définit le nom qu'aura l'onglet dans le navigateur,
  • la balise <xmp> qui permet d'afficher du code html.

Les balises orphelines

Contrairement aux balises paires, les balises orphelines sont seules et peuvent aussi être appelée auto-fermantes. Le terme parle alors mieux puisque la balise ouvre et se ferme toute seule. On ne peut rien mettre d'autres que les éventuels attributs dans ces balises. Voici quelques exemples:

  • la balise <img> qui permet d'insérer une image dans la page,
  • la balise <br> permet surtout un retour à la ligne dans un paragraphe (à noter que celui ci est un cas particulier selon que l'on écrive en HTML5 ou en XHTML mais j'ai pas encore saisi la différence. <br> est valide en html5 mais doit s'écrire comme ça pour le xhtml : <br />. L'espace avant le / n'est pas optionnel.)
  • la balise <link> qui permet de mettre un lien vers un fichier.

Le markdown et l'HTML

le markdown est un langage léger, offrant une syntaxe facile à lire et à écrire. Il peut être lu sans donné l'impression d'avoir été balisé. L'HTML est un langage plus complexe mais permettant de faire presque tout!

En markdown:

Insérer une image ![Alt Text](url) Insérer une citation > Coffee. The finest organic suspension ever devised... I beat the Borg with it. Insérer une vidéo youtube est impossible en markdown

En HTML:

Insérer une image <img alt="nom de l'image" src="adresse.url.de.l.image"> Insérer une citation <cite>citation</cite> Insérer une vidéo youtube <video width="99%" height="540" autoplay loop muted markdown="1"> <source src="/images/posts/send-cancel.mp4" type="video/mp4" markdown="1" > <source src="/images/posts/send-cancel.webm" type="video/webm" markdown="1"> </video>
